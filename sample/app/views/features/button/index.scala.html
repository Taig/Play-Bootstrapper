Basics

To create a button you have to provide a label (may be None) and a Call from your router. All further parameters are optional. @Basics The Style, Size and Icon objects are named like the respective classes in Twitter Bootstrap. Parameters that expect an Option type (in this case label and icon) are backed up by implicit conversions to simplify a button's instantiation. @BasicExamples

Common Use Cases

The labels on the common use case buttons support Play's internationalization tools. Have a look at the configuration files to add your project's language.
